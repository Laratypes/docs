# Giới thiệu

Gói `@laratype/database` cung cấp một ORM (Object-Relational Mapping) mạnh mẽ và linh hoạt, giúp bạn tương tác với cơ sở dữ liệu một cách dễ dàng và hiệu quả trong ứng dụng Laratype của mình.

## Cài đặt

Để sử dụng các tính năng của gói `@laratype/database`, bạn cần cài đặt nó trước:

::: code-group

```sh [npm]
$ npm install @laratype/database
```

```sh [pnpm]
$ pnpm add @laratype/database
```

```sh [bun]
$ bun add @laratype/database
```

:::

## Cấu hình

Bạn có thể chỉnh sửa cấu hình database tại file `config/config.ts`
